Ringkasan — Cara Kerja Internet

Internet adalah jaringan komputer global. Saat Anda membuka sebuah alamat, perangkat meminta alamat IP melalui DNS, lalu terhubung ke server melalui protokol (HTTP/HTTPS). Server mengirimkan kembali file yang dirender browser.

HTTP & HTTPS

HTTP: protokol untuk transfer dokumen web. HTTPS: HTTP + TLS (enkripsi).

Nama Domain

Nama domain adalah alamat mudah dibaca (mis: contoh.com) yang dipetakan ke alamat IP melalui DNS.

Hosting

Hosting menempatkan file situs di server yang selalu online agar dapat diakses lewat internet.

DNS

DNS menerjemahkan nama domain menjadi alamat IP. Sistem ini terdistribusi (root > TLD > authoritative).

Browser

Browser meminta, menerima, dan merender konten. Subsystem utamanya: networking, parser, render engine, JS engine.
